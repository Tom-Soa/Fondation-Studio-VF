Les projets sont maintenant geres via Sanity CMS.
Sanity CMS — Realisations

Ajoutez, modifiez et supprimez vos projets directement dans le studio Sanity. Chaque projet supporte : image native, categorie, description courte, description modale, URL site, ordre et statut (visible / masque / archive).

  • Ajouter un projet : bouton "+" dans Sanity Studio
  • Masquer un projet : champ Statut → "Masque"
  • Supprimer : icone corbeille dans le document
  • Ordre : champ "Ordre d'affichage" (entier, tri croissant)
